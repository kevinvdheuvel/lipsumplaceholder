Welcome to the Lorem Ipsum placeholder for web-developers

Lipsum placeholder is an easy-to-use lightweight jQuery-script to easily fill any element with random Lorem Ipsum text on your website. It works on all browsers that support Javascript.

Getting started

My main objective was to make it super simple and easy-to-use because if it wasn't you could just go to the lorem ipsum website and copy some text.

To get this script working you have to import the script after jQuery because it uses jQuery ;-)

<script src="//ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script>
<script src="//lipsumplaceholder.com/lorem.min.js"></script>

Then all you need to do is add the class "lorumText" to any HTML element. Then add the attribute "numberWords" to tell the script how many words you would like to use and that's it!

<p class="loremText" numberWords="50"></p>
<div class="loremText" numberWords="500"></div>

Example

You can copy this code to see the example or click here for the demo page.

<html>
<head>
<title>Lorem ipsum example</title>
</head>
<body>
<!-- Add the class loremText to any element and set numberWords to any value -->
<p>Here are 50 random Lorem ipsum words inside a paragraph</p>
<p class="loremText" numberWords="50"></p>
<p>Here are 500 random Lorem ipsum words inside a div</p>
<div class="loremText" numberWords="500"></div>

<!-- load jQuery -->
<script src="//ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script>
<!-- load lipsum placeholder script -->
<script src="//lipsumplaceholder.com/lorem.min.js"></script>
</body>
</html>

How does it work?

The script will append random words from a array to the element(s) with the class "lorumText". The value that is used in the attribute "numberWords" will be the amount words that will get appended to the element. It will do that for every element with the class "lorumText" and attribute "numberWords".

This is the script that makes the magic happen.

// Make an Array with all the words
var data = [ "Lorem","ipsum", "..." ];
// loop through all elements on the page that contain the class "loremText"
$( ".loremText" ).each(function() {
// read the value from "numberWords"
var numberWords = $(this).attr("numberWords");
// use the value to loop an X amount of times through the code
for (var i = 0; i < numberWords; i++) {
// create a random number between 0 and the length from the array - 1
var arrayPos = (Math.floor(Math.random() * data.length) + 1) - 1;
$(this).append(data[arrayPos].lorem + " ");
};
});
